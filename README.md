# Maruti Jewellers Website ✨

A stunning, fully animated and responsive jewelry e-commerce website for Maruti Jewellers (maruthijewellers.in), featuring advanced CSS animations, custom cursor effects, and immersive user interactions.

## Features

### Visual Effects:
- **Custom Animated Logo** - SVG diamond logo with rotation and glow effects
- **Preloader Animation** - Beautiful loading screen with animated logo
- **Custom Cursor** - Gold-themed cursor with hover effects
- **Floating Particles** - Dynamic background animations
- **Sparkle Effects** - Twinkling diamonds around hero image
- **Parallax Scrolling** - Smooth depth effects on scroll
- **3D Card Tilt** - Interactive product card tilting on mouse move
- **Scroll Animations** - AOS (Animate On Scroll) library integration
- **Gradient Backgrounds** - Modern gradient overlays
- **Smooth Transitions** - Cubic bezier easing throughout

### Interactive Features:

## How to Use

1. Open `index.html` in any modern web browser
2. Navigate through sections using the menu
3. Click on buttons to interact with the site
4. Test responsive design by resizing the browser

## Customization

### Update Images:
Replace Unsplash URLs in `index.html` with your own jewelry images.

### Change Colors:
Modify CSS variables in `style.css` at the `:root` section:
```css
:root {
    --primary-green: #1a4d4d;
    --gold: #c9a86a;
    /* etc... */
}
```

### Update Content:
Edit text, prices, and descriptions directly in `index.html`.
